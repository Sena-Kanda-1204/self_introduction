# Self Introduction Portfolio

自己紹介・スキル・趣味などを紹介するために制作した、ポートフォリオサイトです。

HTML / CSS / JavaScriptを用いて制作し、レスポンシブ対応やコンポーネント化など、保守性を意識した設計を行っています。

## URL

https://Sena-Kanda-1204.github.io/self_introduction/


## 主な機能

- 自己紹介・プロフィール
- スキル紹介
- 趣味・好きな作品の紹介
- お問い合わせページ
- レスポンシブデザイン
- ハンバーガーメニュー
- ヘッダー・フッターの共通コンポーネント化

## 使用技術

- HTML5
- CSS3
- JavaScript (ES6)

## 工夫した点

- JavaScriptによるヘッダー・フッターの共通化
- ページごとのCSSを分離し、保守性を向上
- セマンティックなHTMLを意識したマークアップ
- 画像の遅延読み込み（Lazy Loading）による表示速度の改善
- スマートフォンでも見やすいレスポンシブデザイン
